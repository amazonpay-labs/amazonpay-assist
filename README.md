## Web接客型 / 支払い救済型 Amazon Payとは
「Web接客型 / 支払い救済型 Amazon Pay」とは、Amazon Payの新しい活用方法です。

**Web接客型Amazon Pay**
会員登録ページ、注文者情報入力ページなど、入力フォームに実装しポップアップを表示することで、情報入力の煩雑な手続きから、ECサイトを離脱する寸前のお客様に便利な購入方法を提案することができます。

**Web接客型Amazon Payのイメージ**
![image](https://dkt57ui24rirk.cloudfront.net/image/QA-91-2.jpg)

**支払い救済型Amazon Pay**
お支払い方法選択ページやAmazon Pay以外の決済方法で決済失敗した際にポップアップを表示することで、ユーザの離脱を防ぎ、スムーズな購入体験を提供することができます。

**支払い救済型Amazon Payのイメージ**
![image](https://dkt57ui24rirk.cloudfront.net/image/QA-91-3.jpg)
